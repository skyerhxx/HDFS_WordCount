# HDFS_WordCount
使用HDFS Java API完成HDFS文件系统上的文件的词频统计

---

### 功能

实现了HDFS文件系统上的文件的词频统计 

又添加了 
​      
* 使用自定义配置文件重构代码(3版本)

* 使用反射自定义Mapper(4版本)

* 可插拔的业务逻辑处理等功能(终版本)



### 开发环境

* Hadoop:  hadoop-2.6.0-cdh5.15.1
* JDK: 1.8
* IDE: IDEA
